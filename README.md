# react-redux-material-ui-form


The localized-react-redux-material-ui-form is React/Redux Form boilerplate code with implementation of:

  - Material UI
  - Formik
  - Yup
  - react-localize-redux
  
  
  Install:
  
          yarn install
          
          
 Development: 
 
          yarn start
